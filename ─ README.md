#  NyondoStock Management System

NyondoStock is a Django-based inventory and sales management system developed to help hardware businesses manage stock, suppliers, sales, customers, credits, and reports efficiently.

The system is designed for small and medium-sized businesses that need a simple and reliable way to track daily operations.
